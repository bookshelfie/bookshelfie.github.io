Prelude: Micropython

What is MicroPython?

  • Python for Microcontrollers
  • Tiny, includes a subset of CPython's libraries
  • Runs off chips with 256kB space and 16k RAM

Getting Started with MicroPython

  • Buy an ESP8266 or ESP32 Chip (Available on Amazon / AliExpress)
  • Flash MicroPython firmware
  • Profit???

ESP8266 (NODEMCU)

ESP8266 (NodeMCU)

  • Runs Lua out of the box
  • Can install MicroPython firmware
  • Supports WiFi
  • Costs ~140 Rs

Flashing MicroPython

  • Download Suitable Firmware
  • Setup Virtual Environment
  • pip install esptool
  • Add current user to tty group (linux)
  • Flash firmware

Communicating with the ESP8266: The MQTT Protocol

  • A Lightweight Protocol for <buzzword>IoT</buzzword>
  • Supported by RabbitMQ using the rabbitmq_mqtt plugin
  • Use the mqtt library in MicroPython, queue names should contain slashes.
  • Protip: Use the amqp.topic exchange and replace slashes in the mqtt queue name with dots.
